# 构建

* [Gradle](https://gradle.org/) - 本插件使用Gradle构建
* 本项目采用GPL v3协议开源 任何使用本项目代码的项目 请使用GPL v3协议

GradleWrapper 已内置在本项目中

**视窗(DOS):**

```
gradlew.bat clean build
```

**视窗(PowerShell):**

```
./gradlew.bat clean build
```

**macOS(苹果系统)/Linux:**

```
./gradlew clean build
```

构建工件将会保存至 `./build/libs` 文件夹.
